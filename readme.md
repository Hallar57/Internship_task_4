# Winter Gallery

A front-end clone of the Tooplate "Winter Portfolio" gallery template, built with HTML and CSS.

## Features

- Fixed frosted-glass navigation bar with hover and active-link snowflake accents
- Full-screen fading image slideshow hero section
- "Winter Moments" heading with Cormorant Garamond and Montserrat font pairing

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS animations, backdrop-filter)
- Google Fonts (Cormorant Garamond, Montserrat)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser

No build tools or dependencies required.
